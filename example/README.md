# react-rerender-debugger Playground

This directory contains a standalone Vite + React playground specifically for testing and developing the `react-rerender-debugger` library locally.

## Getting Started

Since this example links directly to your local codebase, any change you make to the source code will reflect here.

1. Navigate to this directory (if you aren't already here):
   ```bash
   cd example
   ```

2. Install the dependencies. This will automatically link the local library module via `file:..`:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and start tweaking! The playground has examples testing both the HOC and Wrapper component approaches.
